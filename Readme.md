# Java Practice Programs

A collection of beginner-to-intermediate Java practice programs covering object-oriented programming, abstraction, inheritance, interfaces, arrays, collections, exception handling, file handling, and simple real-world simulations.

## 📌 About This Repository

This repository contains 15 standalone Java programs. Each file focuses on a specific programming concept and can be compiled and run independently from the command line.

The exercises are designed to strengthen core Java skills through practical examples such as warehouse storage management, student grade filtering, ATM transactions, inventory discounts, file reading/writing, payroll generation, and more.

## 🧠 Concepts Covered

- Classes and objects
- Abstract classes
- Interfaces
- Inheritance and polymorphism
- Encapsulation
- Arrays and `ArrayList`
- Loops and conditionals
- Custom exceptions
- Exception handling with `try-catch-finally`
- File input/output
- Basic data filtering and analysis
